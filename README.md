<h2> Description</h2>
The project is bootsraped with Next.js and Yran

Open the project in:

Dev mode:

```sh
yarn dev
```

Build:

```sh
yarn build
```

Start prod:

```sh
yarn start
```

The project is an Ecommerse web application inspired by Amazon and Myntra.
The project is aimed to demostrate:

- List product items
- Add to fav
- Add to cart
- Checkout flow
- Authentication (with Auth0)
